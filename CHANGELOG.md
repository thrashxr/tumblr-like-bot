# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-02-09

### Added

-  Added graphical user interface (GUI)
-  Automatic like system based on tags
-  Blog blocking system
-  Hourly and daily like limits
-  Working hours settings
-  Content type filtering
-  Detailed logging system
-  Statistics panel
-  Cache system to prevent duplicate likes
-  Turkish character support

### Security

-  Secure API key management with `.env` file
-  `.gitignore` configurations
-  Protection of sensitive data

### Technical

-  Python 3.x compatibility
-  Modern interface based on PySide6
-  YAML-based configuration system
-  Advanced logging with Loguru
-  Multi-threading support
